# 2048

Il gioco 2048 nella versione ncurses.

## Installazione

### Download
Assicurati di aver effettuato il download della repository:
```bash
$ git clone https://github.com/FabioZTessitore/2048.git
```
### Compilazione della versione ncurses
Assicurati di avere le librerie ncurses
```bash
$ sudo apt-get update
$ sudo apt-get install libncurses5-dev
```
```bash
$ make 2048_ncurses
```

## Esecuzione
```bash
$ ./2048
```

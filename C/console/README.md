# 2048

Il gioco 2048 nella versione console.

## Installazione

### Download
Assicurati di aver effettuato il download della repository:
```bash
$ git clone https://github.com/FabioZTessitore/2048.git
```
### Compilazione della versione console
```bash
$ make 2048_console
```
## Esecuzione
```bash
$ ./2048
```
